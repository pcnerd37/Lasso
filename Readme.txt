Lasso

In Visual Studio, right click the solution and go to propties and set to launch both projects at the same time and hit apply and ok and then run the projects.  Swagger will launch in one Window to interact directly with the API.  The Blazor app Lasso will launch in the other.  Navigate to the Users View link on the left nad hit add user to add users.  There is a limit of 3 users per page and when you add more it will add pages.  API limits 10 API requests per 10 minutes.
